## Ex-44 : month of year :  

Write a program to ask the user to enter :  
- Month  

Then print the day as follows : 
- 1 print January  
- 2 Print February  
- 3 print March  
- 4 print April  
- 5 print Mai 
- 6 print June  
- 7 print Jully  
- 8 print August  
- 9 Print September 
- 10 print October  
- 11 print November  
- 12 print December  

Otherwise print "wrong Month" and ask the user to enter the month again.  

### Example inputs :  
11  

Output => November.
