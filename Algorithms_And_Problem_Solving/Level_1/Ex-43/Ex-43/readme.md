## Ex-44 : Day of week :   
 Write a program to ask the user to enter : 

 - Day  


 Then print the day as follows  : 
 - 1 print sunday
 - 2 print Monday 
 - 3 print Tuesday 
 - 4 print Wednesday 
 - 5 print Thursday 
 - 6 print Friday
 - 7 print Saturday  

 ### example inputs : 
 6  

 Outputs => 
 Its Friday.
