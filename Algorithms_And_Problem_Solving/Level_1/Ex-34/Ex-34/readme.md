## Ex-34 : piggy bank calculator :  

Write a prgram to ask the user to enter :  
- pennies, nickels,dimes,quarters, dollars 

Then calculate the total pennies, total dollars and print them on screen .

 - penny=1
 - Nickel=5
 - Dime =10  
 - quarter=25  
 - dollar=100 
	
### Example Inputs : 5,5,5,5,5  

Outputs =>  
705 Pennies  
7.05 dollars
