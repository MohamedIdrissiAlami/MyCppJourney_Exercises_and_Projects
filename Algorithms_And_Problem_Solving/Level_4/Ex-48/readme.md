## Ex-48 : Compare date function :  
Write a program to read date 1 and date 2 and make a function to compare the two dates.  
It should return :  
- `-1` if date 1 is before date2  
- `0`  if date 1 is equal date 2  
- `1`  if date 1 is after date2  
----  
Enter date 1 :  
- Year? 2022  
- Month? 9  
- Day? 22  

Enter date 2 :  
- Year? 2020  
- Month? 5  
- Day? 25  

-> Compare result : `1`.
