## Ex-50 : period length in days :  
Write a program to read a period and calculate length in days.  
____  
Enter period 1 :  
- Please enter start date :  
Year? 2022  
Month? 1
Day ?1  
- Please enter end date :  
Year? 2022  
Month? 2   
Day ?5

-> Period length is : 4 day(s).  
Period length including end date is : 5 day(s).
