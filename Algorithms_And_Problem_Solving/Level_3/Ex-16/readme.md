## Ex-16 : Check sparse matrix :  
Write a program to check if a 3x3 matrix is sparce or not .  
`Sparce matrix : `is a matrix where most of its numbers are `0`.  
### example :  
Matrix1 :  
1 0 3  
0 9 1  
5 8 1  
No matrix1 is not sparce .  

Matrix2 :  
1 0 4  
0 4 0  
8 0 0  
Yes matrix 2 is sparce
